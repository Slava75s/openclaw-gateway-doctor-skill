# OpenClaw Gateway Doctor Skill

Специализированный навык (skill) для Gemini CLI, предназначенный для диагностики и автоматического восстановления сервисов шлюза OpenClaw на Linux (systemd).

## Основные возможности
- **Глубокая диагностика:** Выявление несовпадений точек входа (entrypoints) и ошибок конфигурации.
- **Умное исправление:** Автоматическое исправление путей в unit-файлах systemd, включая корректную обработку обратных слэшей и экранирование.
- **Проверка здоровья:** Верификация состояния сервиса после внесения изменений.

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Slava75s/openclaw-gateway-doctor-skill.git
   ```

2. Добавьте путь к папке в конфигурацию вашего Gemini CLI (`~/.gemini/settings.json`):
   ```json
   {
     "skills_path": ["/абсолютный/путь/до/openclaw-gateway-doctor-skill"]
   }
   ```

## Использование

Активируйте скилл в чате Gemini CLI:
`activate_skill(name="openclaw-gateway-doctor")`

После активации агент получит инструкции по диагностике и сможет «лечить» шлюз, если вы сообщите об ошибках подключения или проблемах с запуском.
